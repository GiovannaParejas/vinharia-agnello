# Vinharia Agnello – Sistema Básico de Gerenciamento de Vinhos

## Descrição
Este projeto consiste em um sistema básico desenvolvido em JavaScript para cadastro e visualização de vinhos artesanais da Vinharia Agnello.

A aplicação permite ao usuário inserir informações por meio de `prompt()`, receber feedback com `alert()` e visualizar os dados organizados no `console.log()`.

## Objetivo da Atividade
Desenvolver um sistema simples utilizando conceitos básicos de JavaScript, incluindo:

- Variáveis e operadores  
- Interação com o usuário (`prompt()`, `alert()`)  
- Exibição de dados no console (`console.log()`)  

---

## Funcionalidades
- Cadastro de vinhos via prompt  
- Coleta das seguintes informações:
  - Nome do vinho  
  - Tipo (Tinto, Branco ou Rosé)  
  - Safra (ano)  
  - Quantidade em estoque  
- Exibição das informações no console de forma clara e organizada  
- Alertas guiando o usuário durante a execução
